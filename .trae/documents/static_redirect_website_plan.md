# 静态重定向网站实现计划

## 项目概述

创建一个可以托管在GitHub Pages上的静态网站，实现以下功能：
- 支持类似 `s.example.xxx/u/+` 格式的短链接（8-12位随机字符）
- 访问短链接后，在3秒后自动跳转到指定的目标网页
- 使用JSON结构化字符串实现功能配置
- 在未跳转之前，不暴露重定向地址的URL
- 不在JSON中暴露源URL，至少需要经Base64加密
- 界面美观，赛博朋克风格，支持日夜间切换

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript
- **部署**：GitHub Pages
- **无后端**：纯静态实现，所有逻辑在客户端执行

## 项目结构

```
/
├── index.html          # 主页
├── u/                  # 短链接处理目录
│   └── index.html      # 短链接重定向页面
├── data/               # 数据目录
│   └── redirects.json  # 重定向配置文件（Base64加密目标URL）
├── css/                # 样式目录
│   └── style.css       # 主样式文件（包含赛博朋克风格和日夜间切换）
└── js/                 # 脚本目录
    ├── redirect.js     # 重定向逻辑（包含Base64解码）
    └── theme.js        # 日夜间切换逻辑
```

## 核心功能实现

### 1. 短链接解析与重定向

- **URL结构**：`s.example.xxx/u/{random-string}`
- **解析逻辑**：
  - 从URL中提取随机字符串
  - 在 `redirects.json` 中查找对应的Base64加密目标URL
  - 解码Base64字符串获取目标URL
  - 显示3秒倒计时
  - 自动跳转到目标URL

### 2. JSON配置文件（Base64加密）

```json
{
  "redirects": {
    "abc123": {
      "target": "aHR0cHM6Ly9leGFtcGxlLmNvbQ==",  // Base64加密的 "https://example.com"
      "created": "2023-04-18T12:00:00Z"
    },
    "def456": {
      "target": "aHR0cHM6Ly9nb29nbGUuY29t",  // Base64加密的 "https://google.com"
      "created": "2023-04-18T12:30:00Z"
    }
  }
}
```

### 3. 倒计时页面（赛博朋克风格）

- 显示当前短链接信息
- 3秒倒计时动画（赛博朋克风格）
- **不显示目标URL**
- 提供手动跳转按钮
- 赛博朋克风格设计：霓虹色彩、故障效果、未来感字体
- 支持日夜间切换

### 4. 安全性措施

- **Base64加密目标URL**：不在JSON中明文暴露目标URL
- 确保JSON文件格式正确，避免注入攻击
- 防止通过浏览器开发工具轻易获取目标URL

### 5. 错误处理

- 处理未找到的短链接
- 处理JSON加载失败
- 处理Base64解码失败
- 处理目标URL无效的情况

## 设计风格

### 赛博朋克风格元素

- **色彩**：霓虹蓝、紫色、粉色、绿色等亮色与深色背景对比
- **字体**：未来感十足的无衬线字体
- **效果**：故障艺术(Glitch)效果、扫描线、霓虹光效
- **布局**：几何形状、网格布局、不规则元素

### 日夜间切换

- **日间模式**：较亮的背景，柔和的霓虹色彩
- **夜间模式**：深色背景，鲜艳的霓虹色彩
- **切换机制**：通过按钮切换，保存用户偏好到本地存储
